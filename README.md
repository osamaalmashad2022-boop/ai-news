# نبض الذكاء AI (AI News Arabic) 🚀

موقع إخباري متخصص في عرض أخبار وتطورات الذكاء الاصطناعي **ملخّصة باحترافية وبدون إخلال بالمصداقية** باللغة العربية، مجاناً 100% وبدون تدخّل يدوي.

---

## 🌟 مميزات المشروع

- **عربي بالكامل (RTL):** خطوط ذاتية الاستضافة (`IBM Plex Sans Arabic` و `Cairo`) مع اتجاه كامل من اليمين إلى اليسار.
- **تصميم داكن عصري (Dark Modern):** ألوان نيونية، بطاقات زجاجية (`Glassmorphism`)، وحركات سلسة بالـ CSS و View Transitions.
- **تحديث تلقائي يومي:** خط إنتاج يجمع بين خلاصات **RSS المجانية** كـ (مصدر الحقيقة) ونموذج **Gemini 2.5 Flash** للتلخيص والتصنيف بدقة وبدون هلوسة.
- **محتوى ثابت وسريع جداً (Static Site):** مبني باستعمال **Astro v5** و **Tailwind CSS**.
- **بحث فوري بدون خادم (Pagefind):** بحث فوري في كامل الأخبار والدليل من طرف العميل.
- **دليل أدوات الذكاء الاصطناعي:** تصفح وتقييم الأدوات مع توضيح حالة التسعير (مجاني / مدفوع / مجاني بمميزات).
- **نشرة بريدية تلقائية بدون كود:** إخراج خلاصة `/rss.xml` جاهزة للربط المباشر مع **Buttondown** أو **Brevo** لإرسال ملخص صباحي للمشتركين.
- **مجاني 100% للأبد:** يعمل على Vercel Hobby و GitHub Actions مجاناً بالكامل.

## 🚀 التشغيل المحلي (Development)

### 1. تثبيت الحزم:
```bash
npm install
```

### 2. إضافة مفتاح Gemini API:
أنشئ ملف `.env` في جذر المشروع وأضف مفتاحك من [Google AI Studio](https://aistudio.google.com/apikey):
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 3. تشغيل سكريبت الجلب والتلخيص:
- **معاينة دون كتابة ملفات (Dry Run):**
  ```bash
  npm run ingest:dry
  ```
- **التشغيل الفعلي وجلب أخبار اليوم:**
  ```bash
  npm run ingest
  ```

### 4. تشغيل خادم التطوير المحلي:
```bash
npm run dev
```
افتح المتصفح على: `http://localhost:4321`

## 🤖 إعداد الأتمتة والنشر (Production Deployment)

### 1. رفع المشروع على GitHub:
- قم بإنشاء مستودع جديد على GitHub وارفعه.

### 2. تعيين Secret في GitHub:
- اذهب إلى **Settings > Secrets and variables > Actions** في مستودعك على GitHub.
- أضف سر جديد باسم `GEMINI_API_KEY` وضع مفتاحك المجاني.

### 3. النشر على Vercel:
- اربط مستودع GitHub بحسابك على Vercel.
- ستقوم Vercel بنشر الموقع تلقائياً عند كل تحديث أو عند تنفيذ GitHub Action اليومي.

---

## 📬 ربط النشرة البريدية (Buttondown / Brevo)

1. افتح حسابك في [Buttondown](https://buttondown.com).
2. اختر ميزة **RSS to Email**.
3. ضع رابط خلاصة موقعك: `https://your-site.vercel.app/rss.xml`.
4. حدد توقيت الإرسال الصباحي. ستقوم المنصة بسحب الأخبار الجديدة وإرسالها للمشتركين تلقائياً.
